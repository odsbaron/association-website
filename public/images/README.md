# 图片资源说明

此文件夹用于存储网站所需的所有图片资源。

## 文件夹结构

```
/public/images/
├── avatars/          # 成员头像图片
├── events/           # 活动相关图片（待创建）
├── projects/         # 项目展示图片（待创建）
└── banners/          # 横幅/海报图片（待创建）
```

## 使用说明

### 1. 头像图片 (avatars/)
- **用途**：成员页面的头像
- **尺寸建议**：256x256 px 或更大（正方形）
- **格式**：.jpg, .png, .jpeg
- **命名规范**：使用拼音或英文名，如 `zhangsan.jpg`
- **引用方式**：`/images/avatars/zhangsan.jpg`

### 2. 活动图片 (events/)
- **用途**：活动页面的展示图片
- **尺寸建议**：1200x630 px（横向）
- **格式**：.jpg, .png
- **命名规范**：`活动名称-日期.jpg`，如 `hackathon-2024.jpg`

### 3. 项目图片 (projects/)
- **用途**：项目展示页面
- **尺寸建议**：800x600 px
- **格式**：.jpg, .png
- **命名规范**：项目英文名，如 `website-redesign.jpg`

### 4. 横幅图片 (banners/)
- **用途**：首页或活动页面的横幅
- **尺寸建议**：1920x600 px（宽屏）
- **格式**：.jpg, .png
- **命名规范**：描述性名称，如 `homepage-hero.jpg`

## 图片优化建议

- 压缩图片以减小文件大小（推荐使用 TinyPNG 或 ImageOptim）
- 单张图片建议不超过 500KB
- 使用 WebP 格式可获得更好的压缩率（可选）

## 注意事项

- 请确保上传的图片有合法使用权
- 避免上传过大的原始图片
- 定期清理不再使用的图片
